vue 설치

--------------------------------
npm install vue 
npm install -g @vue/cli
--------------------------------

설치확인
--------------------------------
vue -V
--------------------------------

vue 프로젝트생성
--------------------------------
*프로젝트 생성을 돕는 cli-init설치
npm i -g @vue/cli-init
--------------------------------

프로젝트생성
--------------------------------
vue init webpack 프로젝트명
--------------------------------

vue 실행
--------------------------------
npm run dev
--------------------------------
